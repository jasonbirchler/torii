# torii
gated audio for norns


very alpha

Requires R engine

Send audio in, each step gates the audio.

Step bar height is set to filter amount for that step

K2 Randomizes pattern
K3 Bypass

E1 BPM
E2 Select Step
E3 Set filter amount for step - set to zero disables step
